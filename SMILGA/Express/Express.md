# What is Express
It is something that is built on top of node and http module.
We can't use express without node.

### If you have wondered why the port 5000
It's because there are specific port for specific purposes.
Such as port 20 to port 80 and port 443(which is for the secure communication)
While in development ports can be arbitary.
Most ports from 1 - 1024 are Taken.
